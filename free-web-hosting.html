Free web hosting

Markus Lehtisalo
Wednesday, 3rd of July, 2024
Espoo, Finland

I recently discovered GitHub Pages, a free way to host websites. The documentation says it's a "static site hosting service that takes HTML, CSS, and JavaScript files straight from a repository on GitHub, optionally runs the files through a build process, and publishes a website."

This site runs on GitHub Pages. So far I'm happy with the service. It was simple to set up and the pages load fast. All you need is a GitHub account and some HTML. You can even connect your own domain for free like I have.

I would recommend GitHub Pages to anyone looking to build a simple static website from scratch.
